Instructions for Compilation:

1. Open ProjectOne.sln present in Carrom_SDL_Project
2. Right click on the ProjectOne on solution explorer and click Properties
3. Inside C/C++, click General
4. Include the include folder present in libraries in Additional Include Directories
5. Inside Linker, click General
6. Include the lib/x64 folder present in libraries in Additional Library Directories
7. Click OK
8. Click Run (Local Windows Debugger) to run the program (game)
